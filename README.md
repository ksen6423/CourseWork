# Проект "Trex(Transaction Explorer)"

## Описание
Проект "Trex(Transaction Explorer)" - это приложение для анализа транзакций.

## Установка

1. Клонируйте репозиторий
git clone https://github.com/ksen6423/CourseWork
2. Установите зависимости
pip install -r requirements.txt


## Использование

1. Зайдите в приложение
2. Выберите и получите информацию по каждой карте (общую сумму расходов и кешбэк)
3. Получите топ-5 транзакций по сумме платежа
4. Получите информацию по курсам валют и стоимости акций
5. Воспользуйтесь поиском переводов физическим лицам
6. Посмотрите информацию по средним тратам в каждый из дней недели за последние три месяца


## Тестирование

Запуск тестов с помощью pytest (установка pip install pytest):
1. python -m pytest (все тесты в проекте)
2. python -m pytest/test_product.py (конкретный файл)

Запуск тестов с помощью unittest (встроенный фреймворк)
1. import unittest
2. python -m unittest discover (выполнение всех тестов в проекте)
